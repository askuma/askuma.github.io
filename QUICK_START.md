# ⚡ Quick Start - Deploy in 5 Minutes

For the impatient. Get your portfolio live in minimal steps.

---

## **Prerequisites (2 minutes)**

1. **Node.js installed?** 
   - Download: https://nodejs.org/ (v16+)
   - Verify: `node --version`

2. **Git installed?**
   - Download: https://git-scm.com/
   - Verify: `git --version`

3. **GitHub account?**
   - Create at https://github.com/signup

---

## **Deploy (3 minutes)**

```bash
# Step 1: Create project
mkdir askutosh-portfolio
cd ashutosh-portfolio

# Step 2: Copy these files into the folder
# - Copy Portfolio.jsx to src/components/
# - Copy package.json to root
# - Copy public/index.html to public/
# - Copy src/index.js, App.jsx, index.css to src/

# Step 3: Install dependencies
npm install

# Step 4: Build and deploy
npm run deploy
```

---

## **Done! 🎉**

Your portfolio is now live at: **https://askuma.github.io**

(Takes 1-2 minutes to appear)

---

## **Next: Customize (5 minutes)**

Edit `src/components/Portfolio.jsx`:

1. **Change name**: Find "Ashutosh Kumar" → replace with yours
2. **Update email**: Find "ashutosh.kumar1089@gmail.com" → replace
3. **GitHub URL**: Replace "askuma" with your username
4. **LinkedIn URL**: Replace with your profile
5. **Experience & Projects**: Update with your own

Then redeploy:
```bash
npm run deploy
```
